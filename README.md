
# Junk-Removal-Competitor-Analysis

## Project Summary

This is a freelance data analysis project that applies exploratory data analysis and data visualization techniques to help a newly established local junk removal business learn from its competitors. The analysis focuses on comparative insights derived from competitor website content and customer reviews from platforms such as Google and Yelp.

## Project Scope

The scope of this project focuses on a comparative analysis of junk removal businesses in the DMV area, companies that are competitors of the client. Analysis will be done on their websites content, google/yelp reviews, and general items related to online presence. The project will produce business insights and suggestions 

This project does not cover any social media analysis, SEO audit analysis, or analysis pertaining to paid advertising.
